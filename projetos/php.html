4. Como Funciona

Este projeto simula um CRUD de produtos utilizando apenas JS no navegador, mas segue a lógica de um sistema PHP com banco de dados:

  • Create: Adicionar novos produtos à lista.
  • Read: Exibir produtos e carrinho.
  • Update: Alterar quantidade ou detalhes (simulado ao re-adicionar).
  • Delete: Remover produtos da lista ou do carrinho.

Permite testar interatividade, categorias, lojas e quantidade, exatamente como um CRUD real.
